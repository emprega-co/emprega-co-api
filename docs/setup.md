# Setup do Ambiente

## Pré-requisitos

- Java 21+
- Maven
- PostgreSQL
- Git
- IntelliJ IDEA ou VS Code

---

## Clonar

```bash
git clone https://github.com/emprega-co/emprega-co-api.git
```

---

## Executar

```bash
mvn spring-boot:run
```

---

controller/
