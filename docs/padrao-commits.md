# Padrão de Commits

Utilizamos o padrão Conventional Commits.

## Tipos

### feat

Nova funcionalidade.

```
feat: criar tela de login
```

### fix

Correção.

```
fix: corrigir autenticação
```

### docs

Documentação.

### style

Alterações visuais.

### refactor

Refatoração.

### test

Testes.

### chore

Configurações e dependências.
